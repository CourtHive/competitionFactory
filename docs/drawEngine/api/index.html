getSeedAssignments

Returns existing seedAssignments for all structures within a draw

Defaults to { stage: 'MAIN', stageSequence: 1 } if { structureId: undefined }

The structure of an assignment object is as follows:

{
  seedNumber: 1,
  seedValue: 1,
  participantId: 'uuid-of-participant'
}
ParametersRequiredTypeDescription
structureIdOptionalstringReturn assignments for a specific structure, identified by structureId
stageOptionalstringLocate structure by stage; used together with stageSequence
stageSequenceOptionalnumberLocate structure by stageSequence; used together with stage

The result is an array of objects which contain seeding details for all structures within the current draw

Object AttributesTypeDescription
structureIdstringunique identifier for draw structure
seedAssignmentsarrayarray of assignment objects
stagestringdraw stage within which structure appears
stageSequencenumberstageSequence within a draw stage
seedLimitnumbereither defined structure seedLimit or number of drawPositions
